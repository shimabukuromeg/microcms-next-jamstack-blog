## このリポジトリについて

[microCMS + Next.jsでJamstackブログを作ってみよう](https://blog.microcms.io/microcms-next-jamstack-blog/) をやってみた

## Getting Started

```bash
$ yarn install
$ yarn build
$ yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
